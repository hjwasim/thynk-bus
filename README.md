# Thynk Bus - Bus Ticket Booking System

This project is a **bus ticket booking system** with **separate backend and frontend applications**.

- **Backend**: Handles trip search, pricing, seat management, booking, and trip cancellation.
- **Frontend**: A mobile application built with React Native and Expo for users to search trips, view seats, and manage bookings.

Both applications are maintained in **separate directories** and communicate via REST APIs.
